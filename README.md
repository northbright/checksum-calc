# checksum-calc

checksum-calc is a program written in [Golang](http://golang.org) which calculates file MD5, SHA-1 checksums.

#### Usage

    -f string
            File to calculate MD5 / SHA-1 hash. Ex: -f='my-cd.iso'
    Usage:
    checksum-calc -f=<file>
    Ex: checksum-calc -f='my-cd.iso'
